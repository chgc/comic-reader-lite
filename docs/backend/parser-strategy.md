# 8comic 來源解析策略

## 目的
在後端代理層解析 8comic 章節頁，輸出前端可直接渲染的圖片 URL 清單。

## 現況策略
1. 發送 HTTP 請求抓取章節 HTML（帶常見瀏覽器 `User-Agent`）。
2. 解析 `<img src="...">` 內容。
3. 補充掃描 HTML 中直接出現的圖片 URL（jpg/jpeg/png/gif/webp）。
4. 去重並輸出。

## 後續優化方向
- 增加章節頁特定腳本變數解析（若頁面以 JS 組圖）。
- 增加重試與來源變更告警。
- 增加來源格式檢測，避免誤抓到非漫畫圖片。
